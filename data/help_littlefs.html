• HX3 WiFi
    File Manager

  • On File Manager page, you may upload and manage files present on WiFi interface flash memory. These files are needed by WiFI interface's built-in file server to display data in your browser.

    Click on a file name link to view file in browser.

    Click on Download to download file to your PC.

    Click on Delete to delete file from flash memory. Warning: Deleting files may lead to unwanted results.

    Click on Select File... button (lettering depends on browser language) to select a file to be uploaded to flash memory. Warning: Overwriting files may lead to unwanted results.

    Click on Upload button to finally upload opened file to flash memory. Warning: Overwriting files may lead to unwanted results.

    Click on Format File System button at end of file list to format a corrupted file system. Warning: All files must be uploaded again; start with littlefs.html and style.css files.

    File list:

    style.css - Style sheet for all pages
    littlefs.html - File Manager page
    config.html - WiFi main configuration page
    params.html - Parameter Editor page
    presets.html - Preset Name Editor page
    ccset.html - CC Set Editor page
    help_xxx.html - Help pages as this one
    params_xx.txt - Tables containing lists of editable parameters used by Parameter Editor page
    timeout.html, getval.html - auxiliary files for future use

    If an update of WiFi interface is due, files in data directory provided with update must be uploaded to WiFI interface flash memory as decribed above.
