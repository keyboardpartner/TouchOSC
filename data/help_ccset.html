    CC Set Editor

  • On CC Set Editor page, you may configure up to 11 HX3 MIDI CC sets (9 predefined/fixed, 2 custom sets). In a CC set, each HX3 parameter (function) has a counterpart consisting of MIDI Channel and MIDI CC number. When this combination is received, HX3 will perform the parameter's function.

    Enter Channel/Control pairs for each desired controller function (see MIDI implementation chart of your instrument). Set Channel to 0 for parameters to be ignored.

    Please note: Decimal notation used. There is no range or duplicates checking of entered values on this simple web interface, so be sure not to exceed parameter ranges and only assign a channel/cc combination once. Entered value will be transferred to HX3 temporarily (may tried out immediately), but will be stored to current CC set only by clicking the "Store active CC Set" button.

    Of the existing CC Sets, those that do not use proprietary MIDI commands are
    available for editing:

    Set 0: NI B4 d3c -  some fixed functions, edit not recommended
    Set 1: Hammond XK - receives NRPN, send CC edit possible
    Set 2: Hammond SK - receives NRPN, send CC edit possible
    Set 3: Versatile - edit possible if necessary
    Set 4: Nord C1/C2 -  edit possible if necessary
    Set 5: VoceDrawbar -  edit possible if necessary
    Set 6: KeyB/Duo -  edit possible if necessary
    Set 7: Hamichord -  edit possible if necessary
    Set 8: KBP/Touchpad -  edit not recommended, Touchpad will no longer work
    Set 9: Custom 1 - fully editable
    Set 10:Custom 2 - fully editable

    The default value range is from 0 to 127 for drawbars and potentiometers, but
    also for switches (0 = Off, 127 = On). For some functions, the value range is
    smaller, which you can set in the Max. column.

    In the column Mode you choose how received MIDI data is interpreted:

    0: Limit to min/max: Values above the range are limited to the respective end value.
    1: Scale to max: If a smaller Max. value range is set, e.g. 0 to 5, incoming
    values 0 to 127 are scaled to the set range.
    2: Ignore out of range: Values outside the Max. range are ignored.
    3: Invert value: Received values are inverted, i.e. 0 becomes 127, 127 becomes 0.
    4: Toggle value: The function is switched on and off alternately each time the
    control element is operated.

    HX3 will send MIDI data according to selected CC set when MIDI Option parameter #1369 is set to 0 (default). Hammond CC sets do extensively use NRPN for receive. These values will be sent as MIDI CCs, not as NRPNs.
