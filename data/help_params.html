• HX3 WiFi
    Parameters

  • On Manage Board Parameters page, you may configure a couple of HX3 parameters including important system settings. Boolean values (e.g. switches/tabs) must be 255 for ON, 0 for OFF, analog values (pots/drawbars) are usually in range 0 to 127. Please note: There is no range checking of entered values on this simple web interface, so be sure not to exceed parameter ranges. Enter desired value and submit the line you changed.

    Please refer to HX3 Editor for a complete list.
  • #Parameter Description
    14962ndDB Select Voice Number Selecting this voice number on a manual will turn on second drawbar set (default 1). If no second drawbar set is present, enter 40.
    1497Generator Model Limit 4 = up to M3/100, 7 = all
    1498CommonPreset Save/Restore Mask Determines which params are saved to Common Presets: Bit 0 (rightmost): Tabs/Knobs/ADSR, Bit 1: GM Voices, Bit 2: Volumes/TrimPots, Bit 3: Vib/Gen/Perc params, Bit 4: Keyboard/Contacts, Bit 5: Split Mode/Split Point/Local Enables, Bit 6: Rotary Setup. Manual Drawbars are always recalled.
    1499Button Mask 0 PL25 Pin Setup (set bit to 0 if button attached, 1 if switch attached): Rightmost bit 0 = Pin 1, bit 1 = Pin 2 etc.
    1500Button Mask 1 PL26 Pin Setup (set bit to 0 if button attached, 1 if switch attached): Rightmost bit 0 = Pin 1, bit 1 = Pin 2 etc.
    1501Audio Taper (log.) Pots Potentiometers (Volume,Gain, Equalizer etc) are audio taper (255) or linear taper (0)
    1502Additional Tab Panel Setup Additional Tab Panels are Switch Inputs, Rightmost bit 0 = Panel I2C Addr. $63, bit 1 = Panel I2C Addr. $64 etc.
    1503ADC Configuration 0=off/module mode, 1=swell pedal only, 2=all ADCs in mk4 MPX mode with no Touch buttons, 3=all ADCs in mk5 external MPX on drawbars mode with Touch buttons enabled
    1504Panel 16 Configuration Panel 0 Row 2 Configuration, 0=On/off, 1=Vibrato/Chorus, 2=OverallPresets
    1505Preset 16 Configuration if only one Preset 16 is used: 0=normal (Upper or Lower), 1=Split Upper/Lower, 2=Split Upper/Common, 3= Split Lower/Common
    1506Pedal Drawbar Configuration 0 = 2 Drawbars, 1 = 4 DBs, 2 = 12 DBs; if 0 or 1, all 12 pedal drawbars will be auto-adjusted according to setting
    1507ADC Scaling for analog Drawbar Input control voltages (100 = 100%, more for limited potentiometer voltage range)
    1508Latching Preset Keys Use latching-type keys (as on vintage Hammond consoles) connected to Preset12-2
    1509Wrap Menus Wrapping Menus if 255, Limit to Start/End if 0
